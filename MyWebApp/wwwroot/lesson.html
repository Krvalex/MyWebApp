Урок 1: Как запустить первое приложение на Java

Введение

В этом уроке мы рассмотрим, как создать и запустить ваше первое приложение на языке программирования Java. Этот процесс включает в себя установку необходимого программного обеспечения, написание простого кода и его запуск. Этот урок предназначен для начинающих, которые только начинают изучать Java, и не требует предварительных знаний.

Прежде чем начать, убедитесь, что у вас есть доступ к компьютеру с установленной операционной системой (Windows, macOS или Linux). Мы начнем с установки Java Development Kit (JDK), который необходим для разработки и запуска Java-приложений.

Шаг 1: Установка JDK

Java Development Kit (JDK) — это набор инструментов, необходимых для разработки и запуска Java-программ. Выполните следующие шаги:

  1. Перейдите на официальный сайт Oracle или используйте OpenJDK.
  2. Скачайте версию JDK, подходящую для вашей операционной системы.
  3. Следуйте инструкциям установщика, чтобы завершить установку.
  4. Проверьте установку, открыв терминал (или командную строку) и введя команду: 
    java -version
    Если установка прошла успешно, вы увидите версию установленного JDK.

Шаг 2: Написание первого приложения

Теперь, когда JDK установлен, давайте создадим простое приложение, которое выведет на экран сообщение "Hello, World!".

  1. Откройте любой текстовый редактор (например, Notepad, VS Code, IntelliJ IDEA).
  2. Создайте новый файл и назовите его HelloWorld.java. Обратите внимание, что имя файла должно совпадать с именем класса в коде.
  3. Скопируйте следующий код в файл: 
    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  4. Сохраните файл.

Шаг 3: Компиляция и запуск программы

Теперь нам нужно скомпилировать и запустить наш код. Для этого выполните следующие шаги:

  1. Откройте терминал (или командную строку) и перейдите в папку, где находится файл HelloWorld.java. Например: 
    cd C:\Users\YourName\Documents\JavaProjects
  2. Скомпилируйте файл, введя команду: 
    javac HelloWorld.java
    Если ошибок нет, в папке появится файл HelloWorld.class.
  3. Запустите программу, введя команду: 
    java HelloWorld
    Вы должны увидеть сообщение Hello, World! в терминале.

Объяснение кода

  • public class HelloWorld — определяет класс с именем HelloWorld. Имя класса должно совпадать с именем файла.
  • public static void main(String[] args) — это точка входа в программу. Java начинает выполнение программы с метода main.
  • System.out.println("Hello, World!"); — выводит текст "Hello, World!" на экран.

Возможные ошибки

Если что-то пошло не так, проверьте следующее:

  • Убедитесь, что имя файла совпадает с именем класса (включая регистр).
  • Проверьте, что JDK установлен корректно, и команда java -version работает.
  • Убедитесь, что вы находитесь в правильной папке в терминале.
  • Проверьте синтаксис кода на наличие опечаток.

Заключение

Поздравляем! Вы только что создали и запустили свое первое приложение на Java. Это первый шаг в изучении языка программирования Java. В следующих уроках мы углубимся в синтаксис языка, работу с переменными, методами и классами. Помните, что практика — ключ к успеху, поэтому не бойтесь экспериментировать с кодом.
